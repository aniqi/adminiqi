# Adminiqi - web-site administarator panel
# version 0.0.1



## Add in config/app.php 
```
	'providers' => [
	...
		/*
       		* Package Service Providers...
    	*/

        Aniqi\Adminiqi\AdminiqiServiceProvider::class,

    ...
```

## Install

Via Composer

``` bash
"aniqi/adminiqi": "master-dev"
```



## PHP Artisan command

``` bash
php artisan vendor:publish
```
